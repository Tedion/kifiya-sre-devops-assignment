# Architecture Diagram Instructions

Please create an architecture diagram file named `architecturediagram.png` showing:

## Components to Include:
- **Ansible Control Node**: Central server running Ansible
- **Target Nodes**:
  - Web servers (web1, web2, web3)
  - Database servers (db1, db2)
  - Application servers (app1, app2)
- **SSH Connections**: From control node to all target nodes
- **Network Topology**: Show network segments/VPCs if applicable

## Tools for Creating:
- draw.io (https://app.diagrams.net/)
- Lucidchart
- Excalidraw
- PlantUML
- Visio

## Diagram Style:
- Use clear labels
- Show data flow (playbook execution)
- Indicate roles applied to each server group
- Include security boundaries if applicable
